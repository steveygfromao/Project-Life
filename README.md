# GameSterria

Infinite 2d Cellular Automate generated landscape game (think Terraria)

Written in Java using OpenGL

Author: Steve Green

# usage

Unzip my-gdx-game-core.zip into the cloned folder (this is a big file hence why it was zipped up)
